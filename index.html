Living Lots

This is the user guide for Living Lots, the vacant lot organizing platform by 596 Acres.

Getting started

This guide assumes you have a functioning implementation of Living Lots. If you would like to install a new Living Lots site, please start with the django-livinglots README (or ask a programmer to look at the it if you're not one).
